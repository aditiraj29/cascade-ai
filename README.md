# 🚀 Cascade AI — Network Failure Prediction Dashboard
##Live Demo https://cascade-ai-blue.vercel.app/

## 📌 Overview

Cascade AI is an advanced interactive dashboard designed to simulate and analyze **cascade failures in complex network systems**. It visualizes network topology, predicts failure risks, and provides AI-driven insights into system resilience.

This project demonstrates concepts from **Network Science, Machine Learning, and Data Visualization** in a modern web-based interface.

---

## 🎯 Key Features

* 🔗 **Dynamic Network Visualization**

  * Interactive graph rendering using canvas
  * Scale-free network generation (Barabási-Albert model)

* 📊 **Risk Prediction System**

  * Cascade failure probability estimation
  * Network resilience scoring

* ⚡ **Cascade Simulation**

  * Multi-layer failure propagation
  * Real-time node state updates

* 🧠 **AI-Based Insights**

  * Explanation of network vulnerabilities
  * Identification of critical nodes

* 📈 **Analytics Dashboard**

  * Risk vs Resilience chart (Plotly)
  * Network metrics (betweenness, clustering, etc.)

* 🎨 **Modern UI**

  * Dark-themed futuristic design
  * Responsive layout

---

## 🧠 How It Works

1. A network is generated using a **scale-free model**
2. Nodes are assigned properties like:

   * Degree
   * Betweenness centrality
   * Load
3. ML-inspired logic estimates:

   * Cascade risk
   * Network resilience
4. Simulation propagates failures based on:

   * Node connectivity
   * Failure thresholds
5. Insights highlight:

   * Critical hubs
   * Vulnerability patterns
